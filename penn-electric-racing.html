Penn Electric Racing Projects

Contributing to Penn's award-winning Formula SAE Electric Vehicle team through innovative hardware solutions.

Hardware-in-the-Loop (HIL) Testing System

Project Overview

Developed a sophisticated testing system that enables Hardware-in-the-Loop (HIL) testing to validate critical vehicle functions. The system specifically ensures proper operation of crucial components like the brake sensor, Ready-To-Drive-Sound (RTDS), and Super Capacitor (SCAP) systems. This testing adapter is instrumental in validating critical functions that interface between both the Power Control Module (PCM) and the Data Acquisition Dashboard (DAQDASH).

Hardware-in-the-Loop Testing Board

HIL System Implementation

Key Features

  • Real-time simulation of vehicle sensor inputs and actuator outputs
  • Comprehensive testing environment for embedded software validation
  • Automated test sequence capabilities
  • Data logging and analysis tools
  • Configurable fault injection for robustness testing

Advanced PCB Design

Project Overview

Designed and developed critical PCBs for the Formula SAE Electric Vehicle, focusing on reliability, performance, and safety. These boards are essential components of the vehicle's electrical system.

Custom PCB Design for Vehicle Systems

Technical Details

  • Multi-layer PCB design using Altium Designer
  • Implementation of safety-critical circuits
  • Integration with vehicle CAN bus network
  • EMI/EMC considerations and testing
  • Thermal management and power distribution
